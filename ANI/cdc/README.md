# 项目说明

- 项目需维护时，本地代码提交到本仓库后，还需上传至服务器进行部署。


#### 登录线上部署服务器
   ```
   ssh gd@123.57.36.185 -p 18341
   ```

p.s. 有登录需要请先找op同学加一下权限

#### 服务器目录结构

路径|说明|
|:--|:--|
~/customized|git源码
~/www_new|正在运行版本代码（其实没啥用 = =。）
~/www_new_bak|正在运行代码副本（其实也没用 = =b）
/etc/nginx/nginx.conf|nginx配置文件

p.s. 真正线上运行代码由nginx配置文件中路径决定，有变化请更新该路径代码。
